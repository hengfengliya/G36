# 英雄联盟5v5预约对战

一个简单的英雄联盟5v5预约对战HTML页面，支持红蓝两方队员预约功能。

## 功能特点

- 🔵 蓝色方 vs 🔴 红色方 5v5对战布局
- 👤 用户昵称格式验证 (昵称##数字)
- ⏰ 自动显示加入时间
- 💾 本地存储数据持久化
- 📱 响应式设计，支持移动端
- 🎮 英雄联盟风格UI设计

## 使用方法

1. 打开 `index.html` 文件
2. 输入召唤师名称 (格式: 昵称##数字，例如: 小明##1234)
3. 选择加入蓝色方或红色方
4. 点击"加入战斗"按钮

## 部署到Vercel

1. 将项目推送到GitHub仓库
2. 在Vercel中导入GitHub仓库
3. 自动部署完成

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- LocalStorage API

## 预览

项目包含了完整的响应式设计，在桌面端和移动端都有良好的显示效果。
